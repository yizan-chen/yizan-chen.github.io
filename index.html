Hawaii Alert

This is a group project in the Software Design class. This project was inspired by the false nuclear missile alert event in Hawaii, which created major panic and fear in the morning. The mistake was blamed on the bad UI design of the alert system and took them hours to sent clarification out after the false alert. So, our objective is to design a UI that will minimize human error and include to maximize response time to unexpected situations. Our end project was built on browser-based languages and agreed on a simple clear design. We did unit testing and implemented mobile text notification and email notification alert. I gained experience in product testing, UI design, and code development process.

Keyword: Web, font-end, GUI, JavaScript, HTML, API, Testing

Support Vector Machine

This is a project in the Machine Learning class. The focus of the class was on Support Vector Machine, to learn how it works and different variation of it. SVM(Support Vector Machine) is an algorithm used to separate data into groups using a given training data. It is a very efficient way to automatic the process of sorting large amounts of data. But there are many variables in the algorithm and no one fits all model. So, for my project, I compared different degrees of kernels and log the effect of overfitting and underfitting the data. I learned how to apply cross-validation to find the best degree for my human-generated data.

Keywork: Python, SVM, Data analysis, Matlab
