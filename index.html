Hawaii Alert

Front-end heavy web group project. In case of disaster events such as tsunami, the user will select the appropriate message to be sent to the public. The goal of this project is to design and develop a UI/UX to minimize human error and avoid unnecessary public fear. Email and mobile text alert are used to send the message.
